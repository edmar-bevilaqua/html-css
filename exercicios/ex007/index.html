Diferenças hoje entro o HTML e o CSS

O HTML hoje se foca muito mais em semântica (significado) e não mais em forma, isso ficou para o CSS hoje.

Por exemplo se eu quiser um endereço, utilizadei o (adress), como apresentado abaixo:

Av. Pastor Luiz Antônio Rodrigues da Luz, n103 - Guaíba - RS

Outra coisa, se eu quiser trocar a coloração de fundo, não é mais utilizado o HTML, nós precisamos apenas chamar o (style) do CSS, como você pode ver no código deste site.
