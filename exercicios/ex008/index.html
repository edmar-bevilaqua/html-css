Esta é a segunda forma de ser realizada, com a tag <em>.


Extra: Marcação com Marca Texto

É possível também passar o marca texto no texto, com mostrado aqui;
Aqui nós temos também uma marcação, so que de outra cor, e isso só é possível usando CSS, pois o HTML se ocupa da semântica e do conteúdo do texto, não dos estilos e cores.


Texto Deletado

Texto deletado tem uma linha no meio (<del>), servindo para dizer que ele esteve presente, mas deve ser desconsiderado.


Texto Inserido

Texto inserido (tag <ins> - "inserted")possui uma linha em baixo, também sendo chamado de sublinhado (<u> - "underline"), porém o <ins> é mais utilizado por ser semântico, e significar um texto inserido.


Texto Sobrescrito e Subscrito

Sobrescrito: ax2 + bx + c. Usado muito em fórmulas, com a tag <sup>
Já o texto subscrito é aquele pequenininho, usado muito em elementos químicos: C6H10O5, usando a tag <sub>.
