Quando queremos mostrar uma ou mais linhas de código, como window.document.getElementById('teste') utilizamos a tag <code>
Como demonstrado agora: window.document.getElementById('teste'), pois assim deixamos as letras monoespaçadas, ou seja, todas as letras com o mesmo espaçamento. Também podemos escrever um trecho de código com a mesma tag, como mostrado abaixo:
